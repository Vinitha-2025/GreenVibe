🌿 GreenVibe - Plant Store Website

GreenVibe is a modern and responsive plant store website built using HTML and Tailwind CSS. The website promotes green living by showcasing a variety of plants, best-selling products, and plant care services.

🚀 Features
Responsive design for mobile, tablet, and desktop devices
Attractive hero section
About Us section highlighting services
Best Sellers product showcase
Newsletter subscription section
Clean and professional footer
Built using Tailwind CSS
